
# IdeaBox

IdeaBox allows users to share and comment on shared ideas.
~~~Python
	class Vote(object):
    count = 0
    def __init__(self, up, down):
        self.up = up
        self.down = down


    def vote(self):
        if self.up == True:
            count += 1
        else:
            count -= 1
~~~
